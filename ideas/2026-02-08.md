# An idea on alignment of 2D visual tokens with 1D markdown

## Context

We have a document image with OCR annotation.

```pseudo
page: Image
  where Image = np.array with shape (height, width, 3)
  
annotation: list[Fragment] where
  Fragment = struct {
    bbox: Box,
    content: str,
  }
```

```pseudo
markdown: Markdown where
  Markdown = struct {
    markdown: str,
    embedded_images: struct {
        path: Path,
        image: Image,
    }
  }
```
